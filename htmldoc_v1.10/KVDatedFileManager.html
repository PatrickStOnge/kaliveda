class KVDatedFileManager

KVDatedFileManager

Handles a set of different versions of files with the same base name and a timestamp

Read all versions of file with a given basename in directory and put them in a list sorted according to their timestamps Environment variables ($KVROOT) and other special symbols ('~') can be used in the directory name.
