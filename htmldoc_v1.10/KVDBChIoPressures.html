class KVDBChIoPressures: public KVDBRecord

             Class KVDBChIoPressures
 This class handles a set of parameters for ChIo pressures.
 It can read/write in from/to a stream with following format :
 ChIos 2_3    p1
 ChIos 4_5    p2
 ChIos 6_7    p3
 ChIos 8_12   p4
 ChIos 13_17  p5
